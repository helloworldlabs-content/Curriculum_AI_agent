# 06.Evaluation

이 폴더는 `05.Advanced_RAG`에서 만든 RAG 결과를 평가하기 위한 전용 코드 모음이다.

평가 항목:
- `06_3.RetrievalEval.py`: Precision@k 기반 검색 품질 평가
- `06_4.FaithfulnessEval.py`: 생성 결과가 검색 문맥에 근거하는지 평가
- `06_5.CoverageEval.py`: 사용자 요구사항이 결과에 얼마나 반영됐는지 평가
- `06_6.RuleEval.py`: 커리큘럼 구조와 시간 규칙이 맞는지 검사
- `06_7.RunEval.py`: 위 평가를 한 번에 실행하는 통합 스크립트

예시 실행:

```powershell
python 06.Evaluation/06_7.RunEval.py --testset 06.Evaluation/06_8.TestsetTemplate.json
```

기본 실행은 `md` 리포트만 저장한다.
JSON 원본도 같이 저장하고 싶을 때만 `--json-output` 옵션을 사용한다.

```powershell
python 06.Evaluation/06_7.RunEval.py --testset 06.Evaluation/06_8.TestsetTemplate.json --json-output 06.Evaluation/raw_report.json
```

LLM Judge 평가를 빼고 로컬 규칙/검색 평가만 돌리고 싶을 때:

```powershell
python 06.Evaluation/06_7.RunEval.py --testset 06.Evaluation/06_8.TestsetTemplate.json --skip-llm-judges
```
