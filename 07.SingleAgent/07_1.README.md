# 07.SingleAgent

이 폴더는 `05.Advanced_RAG`의 생성 파이프라인과 `06.Evaluation`의 평가 파이프라인을
하나의 에이전트 흐름으로 묶은 예시 코드다.

Single Agent가 하는 일:
- 대화 메시지 또는 구조화된 입력을 받는다.
- 필요하면 수집 프롬프트를 이용해 `CollectedInfo`를 추출한다.
- RAG 체인으로 커리큘럼을 생성한다.
- Retrieval / Faithfulness / Requirement Coverage / Rule-based 평가를 한 번에 수행한다.
- 최종 결과를 한국어 마크다운 리포트로 저장한다.

파일 구성:
- `07_2.AgentSchemas.py`: Single Agent 입력/출력 스키마
- `07_3.AgentHelpers.py`: `05.Advanced_RAG`, `06.Evaluation` 모듈을 재사용하기 위한 공통 로더
- `07_4.SingleAgent.py`: 실제 에이전트 본체
- `07_5.RunSingleAgent.py`: 실행용 CLI 스크립트
- `07_6.InputTemplate.json`: 입력 예시

예시 실행:

```powershell
python 07.SingleAgent/07_5.RunSingleAgent.py --input 07.SingleAgent/07_6.InputTemplate.json
```
